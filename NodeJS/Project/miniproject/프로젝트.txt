조원 : 박성희 , 정지욱, 오종훈, 이민지
주제 : 쇼핑몰 (어떤 쇼핑몰인지 정해 지지 않음)
         (예를 들어 그냥 가방 쇼핑몰로 하고, 백팩 / 크로스백 / 에코백 등등으로 나눠도 될듯함.
 
기능구현 : 
 
관리자 page /  고객 page 따로 구현. => 메인페이지로 들어가고 로그인을 하고 나서 관리자가 로그인 했으면 관리자 페이지로 이동 할 수 있는 버튼
                                                              고객은 버튼이 안보임.
 
관리자 : 1) 상품등록(등록, 조회, 삭제, 수정) 
        2) 회원관리 - 회원 정보 보기, 블랙리스트 설정
 
고객 : 1) 상품 구매
          2) 구매 내역 보기
          3) 장바구니 담기
          4) 회원가입 / 로그인 [최초회원가입시 : 포인트지급]
 
리뷰게시판 : 게시글 CRUD
                    게시글 전체 목록 보기
                    특정 게시물보기,
                    검색 or 페이징
 
상품 관련 : 상품 리스트 보기, 상세 보기, 회원은 상품에 댓글달기, 검색 or 페이징
 
DB     : 몽고DB / MYSQL 
 
각각의 유효성 검사는 모두 필요함.
